#SHN

## Requirements
* node.js - version > 0.8
* npm
* grunt


## Develop
* checkout git rep
* open terminal and cd folder
* run command: npm install
* run command: grunt
* open webbrowser go to url http://localhost:9001
* start develop

## Release
* checkout git rep
* open terminal and cd folder
* run command: npm install
* run command: grunt build
* move files from dist folder to live enviroment
